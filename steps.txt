# Install Scoop in windows
- Install Scoop Package Manager

```powershell
Invoke-Expression (New-Object System.Net.WebClient).DownloadString('https://get.scoop.sh')
```
- install jdk

```powershell
scoop search jdk
scoop bucket add java
scoop install oraclejdk
```

java --version
